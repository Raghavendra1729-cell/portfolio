# Raghavendra Portfolio

A minimal, content-first portfolio built with Next.js App Router, TypeScript, and MongoDB.

This repository powers:
- A public portfolio website (home, about, projects, experience, skills, achievements, contact).
- A protected admin dashboard for editing portfolio content.
- API routes for content retrieval, admin CRUD operations, uploads, and profile/stat sync.

## Quick start

1. Install dependencies:

```bash
npm install
```

2. Create `.env.local` (see [docs/SETUP.md](docs/SETUP.md)).

3. Run development server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Quality checks

```bash
npm run lint
npm run build
```
