# Features

## Public site
- Minimal dark UI with consistent section rhythm and readable typography.
- Responsive navbar with active states and mobile menu.
- Project listing and detail pages with links and tech stack context.
- Experience, skills, achievements, and contact sections.
- Empty/fallback states when content is unavailable.

## Admin dashboard
- Session-based admin access using secure httpOnly cookie.
- CRUD operations for all content collections.
- Validation for field lengths, URLs, and structured inputs.
- Upload API integration for media assets.

## Engineering basics
- Strict TypeScript typing and collection-specific normalization.
- Server-side data loading for page content.
- Metadata and canonical URLs for SEO baseline.
- Revalidation after admin content mutations.
